# Agent Handoffs

This directory records implementation and validation handoffs for the
`standardize-effect-package-contracts` migration. Each subagent owns a
disjoint slice from `agent-migration-plan.md` and should write one handoff file
named after its assigned role.

The coordinator should preserve these files during integration. They are
working migration evidence until the change is complete, then they may be
archived with the OpenSpec change.
